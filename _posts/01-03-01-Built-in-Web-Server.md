## وب سرور داخلی {#builtin_web_server_title}

از پچپ 5.4 به بعد می‌توانید بدون نصب و تنظیم یک وب سرور شروع کنید به یاد گرفتن پچپ.
برای اجرا کردن سرور دستور زیر را از شاخه‌ی اصلی پروژه‌تان در ترمینال اجرا کنید:

{% highlight console %}
> php -S localhost:8000
{% endhighlight %}

* [درباره وب سرور داخلی پچپ بیشتر بخوانید][cli-server]
